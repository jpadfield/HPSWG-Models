## Project AI Pormpt
This folder contains the project state document for the HPSWG-Models repository. The project state document is intended to be pasted at the start of a new AI-assisted working session to restore context without re-explanation. It records the current structure of the repository, the TSV format and directive syntax, key design decisions, and outstanding work items.

The most recent version is the highest-numbered file in this folder. Older versions are retained for reference but should not be used to start new sessions.

When significant decisions are made or the repository structure changes, the project state document should be updated before closing the session.
