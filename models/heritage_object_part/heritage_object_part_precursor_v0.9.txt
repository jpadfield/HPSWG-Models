Not yet released to Kartography<br/> <br/> "Based on the the wiki content and how people have worked before there seems to be the need to be able to define an entity to cover a group or class of sections within a painting that a user want to identifiy and discuss as a thing. Includes Support - Medium - Preparatory Layers - Paint Layers - Surface Coatings - Additions - etc. This can be broken down intio individual compoents or layer for analysis or specific description, but people generally need toi write summaries of interpretations based on these groups. It would be good to know if this needs to be a specific form or collection of forms, or whether we can archive this by grouping the individually defined things and discussiong the group. Logically this could work as a single form or process, but from a user point of view we are likely to need to have a selection of forms or porcesses som people can clearly see where to put, for example, their description of all of the paint layers. Moving on from this is potentially a training thing or development of procedures, but we can see."<br/> <br/> Will need separate forms for these parts: paint layer, varnish layer, preparatory layer, other layer of painting not defined above<br/> Will need separate forms for these components: component of paint layer, component of varnish layer, other component<br/> <br/> Support (e.g. stretcher) will need to granularity (e.g. to describe the canvas, type, thread count, other attributes)
